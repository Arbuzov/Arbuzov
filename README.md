## Чем я занимаюсь

- **Home Assistant и IoT** — интеграции для ТВ, счётчиков воды, кофе-машин и кастомных smart-device сценариев.
- **AI / MCP / агентные workflows** — инструменты и эксперименты для AI automation и LLM-driven процессов.
- **DevOps и self-hosted automation** — Docker, Compose, Ansible и удобные интерфейсы для управления сервисами.
